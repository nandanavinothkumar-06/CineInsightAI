# 📌 Overview

CineInsight AI is an end-to-end AI-powered movie review intelligence platform that combines **Machine Learning**, **Deep Learning**, **Natural Language Processing (NLP)**, **Explainable AI (XAI)**, and **Business Intelligence Analytics** into a single web application.

Instead of simply classifying reviews as positive or negative, CineInsight AI explains **why** a prediction was made, visualizes the NLP pipeline, compares multiple machine learning models, analyzes review aspects, detects emotions, and generates actionable business insights.

# 📊 Workflow

1. User submits a movie review.
2. Review undergoes NLP preprocessing.
3. TF-IDF vectorization is applied.
4. Multiple ML models perform prediction.
5. BERT generates contextual prediction.
6. Model consensus is calculated.
7. Explainable AI identifies influential words.
8. Emotion detection is performed.
9. Aspect-based sentiment analysis is executed.
10. Business Intelligence metrics are generated.
11. AI Insights summarize prediction reliability.
12. PDF report is generated.
